# Asahi Docs

Planning and scaffolding documents for the Asahi Android TV streaming app.

## Documents
- `RESEARCH.md` - repo and product research notes
- `ARCHITECTURE_PLAN.md` - high-level architecture direction
- `DOMAIN_CONTRACTS.md` - first-pass domain models and interfaces
- `SCAFFOLDING_PLAN.md` - proposed module/package/file structure
- `IMPLEMENTATION_ROADMAP.md` - execution order for implementation
